#
# Location: C:\Windows\System32\drivers\etc\hosts
#
# How to add these entries:
# 1. Open Notepad as Administrator
# 2. File > Open > Navigate to C:\Windows\System32\drivers\etc\hosts
# 3. Copy all entries below (from "127.0.0.1 web.nem.local" onwards)
# 4. Paste at the end of the file
# 5. File > Save
#
# After adding these entries, you can access services via:
#   http://web.nem.local
#   http://mcp.nem.local
#   etc.
#
# Note: All entries point to 127.0.0.1 because WSL2 mirrored networking
# makes the WSL VM's localhost accessible from Windows.
